# 🇸🇩 Sudan Smart Market

An open-source AI-powered digital marketplace built to empower individuals and small businesses across Sudan.

## Vision

To create Sudan's most trusted open-source marketplace, connecting buyers and sellers through modern technology and artificial intelligence.

## Features

- 🤖 AI-powered product search
- 🛍️ Buy & Sell Marketplace
- 🏪 Business Profiles
- 💬 In-app Chat
- ⭐ Ratings & Reviews
- 📍 Location-based Search
- 🌐 Arabic & English Support

## Technology

- Google AI Studio
- Firebase
- GitHub
- AI Integration
- Modern Web Technologies
