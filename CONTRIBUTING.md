# Contributing to Sudan Smart Market 🇸🇩

Thank you for your interest in contributing to Sudan Smart Market.

We welcome developers, designers, testers, and community members who want to help improve this open-source project.

## How to Contribute

1. Fork the repository.
2. Create a new branch.
3. Make your changes.
4. Test your updates.
5. Submit a Pull Request.

## Contribution Areas

You can contribute in:

- Frontend development
- Backend development
- AI features
- UI/UX design
- Documentation
- Testing
- Bug reports

## Reporting Issues

If you find a problem, please create a GitHub Issue and provide:

- Description of the problem
- Steps to reproduce it
- Screenshots (if available)

## Code Guidelines

- Keep code clean and readable.
- Add comments when needed.
- Follow best practices.

Thank you for helping build a better digital marketplace for Sudan.
