# Security Policy

## Reporting a Security Issue

Thank you for helping keep Sudan Smart Market secure.

If you discover a security vulnerability, please do not create a public issue.

Instead, report it privately by contacting the project maintainers.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes |
| Older versions | No |

## Security Guidelines

- Do not share sensitive information publicly.
- Keep your dependencies updated.
- Report security issues responsibly.

## Response Process

We will review reported vulnerabilities and work on fixes as soon as possible.

Thank you for helping protect the Sudan Smart Market community.
